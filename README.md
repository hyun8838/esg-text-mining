## 📌 프로젝트 배경

LG전자 VOC 텍스트 마이닝 산학협력 프로젝트의 분석 성과를 인정받아, **교수님으로부터 개인적으로 추가 분석을 의뢰받아 수행**한 연구입니다.

수도권 18개 대학 생활과학계열 학과 홈페이지 **1,054건**을 수집해 ESG(Environmental·Social·Governance) 키워드 빈도, TF-IDF, 공동출현 네트워크 분석을 수행한 탐색적 연구입니다.

## 🔍 ESG 키워드 분류 체계

| 범주 | 핵심 키워드 |
|------|-----------|
| **E (Environmental)** | 환경, 지속가능, 생태, 탄소, 그린, 친환경 |
| **S (Social)** | 복지, 공동체, 다양성, 포용, 사회적책임, 지역사회 |
| **G (Governance)** | 투명성, 윤리, 거버넌스, 공정, 책임경영 |

---

## 💡 주요 분석 결과

- **계열별 특성**: 의류계열은 E 키워드(지속가능·친환경 소재) 집중, 아동가족계열은 S 키워드(복지·공동체) 집중
- **대학별 편차**: 수도권 주요 대학 간 ESG 서술 밀도 차이 존재
- **네트워크 특성**: ESG 키워드와 교육·연구·진로 맥락어 간 공동출현 패턴이 계열별로 상이
